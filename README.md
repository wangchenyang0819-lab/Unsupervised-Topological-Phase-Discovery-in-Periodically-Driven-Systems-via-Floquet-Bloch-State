# Floquet Topological Classification for Class AIII (1D)

This folder contains MATLAB code for unsupervised clustering of Floquet topological phases in symmetry class AIII (1D). The method constructs a kernel from the flattened Floquet operator in momentum‑time space, then applies diffusion maps and hierarchical clustering to identify topologically distinct phases. Theoretical winding numbers for the 0‑gap and π‑gap are also computed for validation.

## Files

| File | Description |
|------|-------------|
| `AIIIinvariantsdiffusion.m` | Main script. Computes topological invariants (winding numbers) over the parameter space, performs unsupervised clustering via kernel + diffusion map + hierarchical clustering, and saves the clustering result (`AIII.mat`). Also produces phase diagrams and PCA plots. |
| `blochAIII.m` | Generates Bloch vector components (xx, yy, zz) for a specific set of parameters (e.g., for the 0‑π phase). Saves the data as `kernelAIII0.mat` or `kernelAIIIpi.mat` for later visualization. |
| `fig1.m` | (Not provided here) Plotting script that reads `AIII.mat`, `kernelAIII0.mat`, and `kernelAIIIpi.mat` to produce Fig. 1 in the paper. |

## Data files

- `AIII.mat` – Generated by `AIIIinvariantsdiffusion.m`. Contains:
  - `e`: diffusion map eigenvalues (used for spectral gap analysis)
  - `score`: first three principal components from PCA
  - `tpp`: topological phase label matrix (1, 2, 3) over the parameter grid
  - `B`: parameter values (γ₂, γ₃) in units of π/T
  - `f`: cluster assignment matrix for plotting
- `kernelAIII0.mat` / `kernelAIIIpi.mat` – Generated by `blochAIII.m`. Contain `xx`, `yy`, `zz` (Bloch vector components in (k, t) space) for a representative phase.

## Requirements

- MATLAB (R2020b or later recommended)
- Parallel Computing Toolbox (optional, not required for these scripts)

## Usage

### 1. Compute topological invariants and perform clustering

Run `AIIIinvariantsdiffusion.m` to:
- Scan the 2D parameter space (γ₂, γ₃) with resolution `NN × NN`.
- For each parameter point, compute the Floquet evolution operator and extract winding numbers (0‑gap and π‑gap).
- Classify the phase into three types: Phase I (1,0), Phase II (1,−1), Phase III (0,−1).
- Then perform kernel‑based clustering on a grid of `grid_size × grid_size` samples (default 16×16) using:
  - Floquet eigenstates in momentum‑time space
  - Similarity kernel with hyperparameter `epsilon`
  - Diffusion map embedding
  - Hierarchical clustering with number of clusters determined by eigenvalues close to 1
- Save the results to `AIII.mat`.

```matlab
>> AIIIinvariantsdiffusion
The script will output two figures:

The theoretical phase diagram (based on winding numbers)

The clustering result (colored points in parameter space)

2. Generate Bloch vector data for a specific phase
Edit blochAIII.m to set the coupling parameters (A1, A2, A22, g) to values corresponding to the desired topological phase (e.g., 0‑phase or π‑phase). Then run the script to compute the time‑evolved Floquet‑Bloch states and the Bloch vector components (xx, yy, zz) in (k, t) space.

matlab
>> blochAIII
The script saves the result as kernelAIII0.mat or kernelAIIIpi.mat (adjust the filename in the script as needed).

3. Reproduce the paper figures
Run fig1.m (not provided in this folder) to load AIII.mat and the kernelAIII*.mat files, producing the phase diagram, eigenvalue plot, PCA embedding, and Bloch vector visualizations as shown in the manuscript.

Key parameters
In AIIIinvariantsdiffusion.m

grid_size	16	Number of samples per dimension for clustering (total samples = grid_size²)

Output details
AIII.mat

e: eigenvalues of the diffusion map. The number of eigenvalues close to 1 gives the number of clusters.

score: PCA coordinates of the first three principal components.

tpp: matrix of theoretical phase labels (1,2,3) on the fine parameter grid.

B: 2×grid_size² array of (γ₂, γ₃) values in units of π/T.

f: grid_size×grid_size matrix of cluster indices (1,2,3) for each sample point.

kernelAIII0.mat / kernelAIIIpi.mat

xx, yy, zz: N×N matrices of the Bloch vector components at each (k, t) grid point. Used to visualize the linking structure of the Bloch vector on the (k, t) torus.

Notes
The theoretical winding number calculation uses a discrete line integral over momentum. The results are rounded to nearest integer.

The kernel constant epsilon should be chosen to resolve topological differences without over‑splitting identical phases. For class AIII, epsilon = 0.5 works well.

For other symmetry classes (A, D), please refer to the corresponding subfolders.
